# 📁 应用程序错误 - 0xE10005

### 原因

等待进程超时

### 解决方法

1. 检查Steam是否运行
2. 默认浏览器是否存在
3. Steam库存是否拥有**Counter-Strike**游戏
4. Steam启动游戏时**Counter-Strike**可能处在更新、检查云存档、下载内容时,请重新运行
